PyScada MeterBus Extension
==========================

This is a extension for PyScada to support read and write to MeterBus devices.


What is Working
---------------

 - nothing is test


What is not Working/Missing
---------------------------

 - Test with real hardware
 - Documentation

Installation
------------

 - pip install pyscada-meterbus
